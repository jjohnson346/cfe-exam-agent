Section=Computer and Internet Fraud

Stem=In a well-designed internal control system for computer operations, the users of the system should have access to the production program code.

Options=True | False

CorrectResponse=1

Explanation=Users should not have access to the production program code. Computer operators should not perform computer programming. Adequate supervision should be provided by personnel who do not actually perform the work. Users should be involved in the project from design through final testing. All test results should be approved by both data center management and user personnel. There should be an implementation plan for placing the new system into production.See page(s) 1.1844 in the Fraud Examiner's Manual
